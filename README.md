# 🚀 SysLab - Frontend

시스템 내부 동작(State)을 시각화하여 학습하는 'Operating By Operating' 플랫폼의 프론트엔드.

### 🛠 Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **State Management**: Zustand
- **Visualization**: D3.js, React Flow
- **Terminal**: Xterm.js
- **Editor**: Monaco Editor
- **Styling**: Tailwind CSS
